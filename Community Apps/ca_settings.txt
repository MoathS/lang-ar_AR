Hide Deprecated Applications=إخفاء التطبيقات المهملة
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=هذا الإعداد يخفي التطبيقات المهملة من قائمة التطبيقات المتاحة، لكن ستظل موجودة في قائمة التطبيقات المثبتة او سابقة التثبيت
Hide Incompatible Applications=إخفاء التطبيقات غير متوافقة
This setting will hide applications that are listed as being incompatible with your version of Unraid=هذا الإعداد يخفي التطبيقات الغير متوافقة مع نسخة unraid الحالية
Enable Reinstall Default=مكن إعادة تثبيت الإفتراضية
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=هذا الإعداد يمكنك من إظهار زر في النافذة يمكنك من إعادة تثبيت التطبيق بإستخدام القالب الإفتراضي، هذا الإعداد يبدو غير واضحًا احيانا، لكن يستخدم بالغالب اذا اردت تثبيت التطبيق مرتين بشكل متزامن
Enable additional search results from dockerHub=يمكن نتائج بحث إضافية من DockerHub
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=هذا الإعداد يمكن التطبيقات العامة من إظهار نتائج إضافية من DockerHub لكن يجب ان يكون دوكر ممكنّ
Allow CA to send any emergency notifications=يسمح للتطبيقات العامة بإرسال إشعارات طارئة
This setting allows CA to periodically scan your system and alert you to the presence of=هذا الإعداد
1 Known malware and security vulnerabilities within applications=1 وجود مشكلة أمنية في التطبيقات
2 Critical security vulnerabilities within Unraid OS=2 مشكلة أمنية في Unraid
3 Any installed applications which may seriously hinder the operation of your server=3 اي تطبيق يعطل سير النظام بشكل طبيعي
